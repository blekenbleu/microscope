Preparation


Lamp centering
  1. fully PUSH B.F.
  2. turn power ON, set 6V
  3. fully open diaphragms
  4. Place ND filter on the stage; focus 10x objective on it
  5. remove eyepiece; focus lamp filament on objective exit pupil
  6. center filament on exit pupil; replace eyepiece
Orient polarizer
  1. nearly focus 40x objective on ND filter
  2. insert analyzer
  3. remove eyepiece; rotate polarizer for dark cross image:
  • Pull D.F. for diascopic
  • see also polarizing filters
